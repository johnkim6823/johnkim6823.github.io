Projects

Task Offloading Simulation

https://github.com/johnkim6823/EdgeCloudSim

  • Developed a simulation environment for task offloading in Mobile Edge Computing (MEC) using a modified EdgeCloudSim framework.
  • Implemented and evaluated various Deep Reinforcement Learning algorithms (DQN, PPO, A2C) for efficient offloading decisions.
  • Analyzed key metrics such as latency, energy consumption, and task failure rates to assess algorithm performance under different network conditions.
  • Aimed to find optimal policies for offloading computational tasks from mobile devices to edge servers, enhancing overall system efficiency.

Hanium 2022

https://github.com/hanium-22-HF195

  • Developed a feature-based encryption system to ensure the integrity of video data.
  • Implemented a mechanism to reduce computational costs by encrypting only key features to generate a digital signature for verification.
  • Designed a 4-module architecture comprising a logger, server, verifier, and Web UI.
  • Created a lightweight solution suitable for embedded systems and resource-constrained devices (e.g., black boxes, CCTV).
